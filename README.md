# Backend for movie vue app

## Project setup

### Install dependencies
```
composer install
```

### Create .env file
```
cp .env.example .env
```

### Generate key
```
php artisan key:generate
```

### Migrate and seed database
```
php artisan migrate --seed
```

### Compiles and hot-reloads for development
```
php artisan serve
```
